# 5125Cummulative1

This project is a Blazor application that manages a school database. It includes functionality for managing teachers, students, and courses.

## Features

- List all teachers
- View details of a specific teacher
- List all students
- List all courses

## Technologies Used

- .NET 8
- C# 12.0
- Blazor
- MySQL

## Getting Started

### Prerequisites

- .NET 8 SDK
- MySQL Server


## Usage

### Teachers

- Navigate to `/TeacherPage/List` to view a list of all teachers.
- Navigate to `/TeacherPage/Show/{id}` to view details of a specific teacher.

### Students

- Navigate to `/StudentPage/List` to view a list of all students.

### Courses

- Navigate to `/CoursePage/List` to view a list of all courses.

## Project Structure

- `Controllers/`: Contains the controllers for handling HTTP requests.
- `Models/`: Contains the models representing the database entities.
- `Views/`: Contains the Razor views for rendering the UI.
